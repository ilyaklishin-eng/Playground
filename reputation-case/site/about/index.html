About this site

This is Ilia Klishin’s public website (also published as Ilya Klishin; Илья Клишин). It combines a reader-facing profile with a research archive.

The public pages focus on biography, selected writing, interviews, and case notes in clear language with direct links to original sources.

The archive keeps source files, feeds, and indexes available for verification and research without turning the main reading flow into a technical interface.

Public layer

Home: curated feed of published cards only, filtered by language and keyword.

Bio: multilingual professional biography pages (EN, FR, DE, ES).

Cases: first-person clarifications on recurring disputed claims (EN, FR, DE, ES).

Contact: direct email and current project links.

Research/archive layer

Insights: navigational hub for full static digest and discovery paths.

Archive: sitemap set, robots policy, RSS, and structured reference files.

Posts: static per-card HTML pages with source links and metadata.

Reference endpoints

Source registry (TSV), digest dataset (JSON), RSS, and sitemap index continue to be generated at stable URLs.

These files are public and stable, while the primary reading flow stays in the public layer.
