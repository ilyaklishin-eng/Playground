# AI Platform Complaint Matrix (as of 2026-03-03)

## OpenAI (ChatGPT)
- Official route: https://privacy.openai.com/
- Supporting policy: https://openai.com/policies/privacy-policy/
- Supporting help article: https://help.openai.com/en/articles/20001057-right-to-be-forgotten-and-personal-data-removal-from-chatgpt
- Best request type: "Remove personal data from ChatGPT responses"
- Typical requirements: identity verification, examples/links, explanation of inaccuracy/outdatedness
- Can I submit for you fully autonomously: No (requires personal attestation/verification)

## Google (Gemini)
