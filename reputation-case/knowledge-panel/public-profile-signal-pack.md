# Public Profile Signal Pack (for Knowledge Graph reinforcement)

## Goal
Make Google's entity extraction pick consistent, neutral facts from trusted pages.

## Must-have pages
1. Personal site bio page (canonical source)
2. LinkedIn profile
3. X/Twitter profile bio
4. At least one newsroom/author profile page with neutral bio

## Consistency rules
- Same full name string on all pages
- Same one-sentence bio on all pages
- Same city/country and role wording
- No contradictory dates/titles

## One-sentence bio (recommended)
Ilia (Ilya) Klishin is a journalist and media professional based in Lithuania.

## Structured data
- Add Person schema on personal site (name, sameAs, jobTitle, url).
- Add links to profiles in sameAs.

## Review cadence
- Re-check panel monthly.
- Update any outdated profile immediately.
