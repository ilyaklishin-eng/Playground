# Submission Checklist

## Can be done without user identity attestation
- Build URL map and evidence text package.
- Prepare all drafts and per-form text.
- Open official routes and prefill non-attestation text locally.

## Requires user action (blockers)
- Any Google/Bing legal/privacy form requiring personal declaration, identity verification, account session, captcha, or signature.
- Any publisher email/webform that must be sent from personal mailbox for credibility/legal standing.
