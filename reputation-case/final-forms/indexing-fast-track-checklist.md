# Indexing Fast-Track Checklist

Updated: 2026-03-25

## 1) Publish to production (required)

1. Open PR from `codex/nkry-citation` to `main`.
2. Merge PR.
3. Wait for GitHub Pages deploy workflow to complete.

## 2) Check public URLs (required)

1. https://www.klishin.work/
2. https://www.klishin.work/bio/
3. https://www.klishin.work/selected/
4. https://www.klishin.work/posts/
5. https://www.klishin.work/sitemap.xml
6. https://www.klishin.work/rss.xml
7. https://www.klishin.work/robots.txt

## 3) Accelerate indexing (optional but recommended)

1. In Google Search Console: URL inspection -> request indexing for:
- `/`
- `/bio/`
- `/selected/`
- `/posts/`
- `/interviews/`
2. Submit sitemap in Search Console:
- `https://www.klishin.work/sitemap.xml`
3. In Bing Webmaster Tools: submit same sitemap.
4. In Yandex Webmaster: submit the same sitemap if Russian-language branded discovery matters.

## 4) Expected timing

- First crawl signal: typically within 24-72 hours after deployment or submission.
- Branded ranking shifts: usually 1-4 weeks, sometimes longer.
- Broader indexation of post pages: usually 3-14 days, sometimes longer.
