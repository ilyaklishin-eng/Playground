# Reputation Case Workspace

This folder stores evidence, routing strategy, and draft submissions for requests related to harmful or inaccurate search-visible content about Ilya Klishin.

## Structure
- `master-table.md` - central tracker
- `dossiers/` - per-URL mini-dossiers
- `publishers/` - publisher correction and editor's note drafts
- `google/` - Google delisting/outdated-content drafts + official links
- `bing/` - Bing privacy/blocking drafts + official links
- `ai-platforms/` - AI platform personal data request drafts + official links
- `archives/` - archive/cache notes
- `snippets/` - captured snippets and query evidence
- `final-forms/` - ready-to-submit text blocks
